Background Music Files
=====================

This folder should contain the following MP3 files:

1. romantic1.mp3 - Romantic piano music
2. romantic2.mp3 - Sweet melody
3. romantic3.mp3 - Love symphony

You can download royalty-free romantic music from:
- YouTube Audio Library
- Pixabay Music
- Freesound.org
- Zapsplat

Make sure the music files are named exactly as above and are in MP3 format.

Alternatively, you can use online music sources and update the template to use URLs instead of local files.
